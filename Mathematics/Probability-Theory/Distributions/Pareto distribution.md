# Lindy effect

If $T$ is a random variable representing a lifetime,

\begin{align\*}
\mathbb{E}\left[T \mid T > t_0\right] &= \int_\mathbf{R} t f(t \mid t > t_0) \\,\dd{t} \\\\
&= \int_\mathbf{R} t f(t \mid t > t_0) \\,\dd{t} \\\\
\end{align\*}
