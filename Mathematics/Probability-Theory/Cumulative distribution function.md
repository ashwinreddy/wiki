The **cumulative distribution function (CDF)** is a [[function]]

\begin{equation}
F_X(x) = \Pr(X \leq x)
\end{equation}

For a continuous distribution, we can just write then

\begin{equation}
F_X(x) = \int_{-\infty}^{x} f_X(u) \\, \dd{u}
\end{equation}

The [[probability density function]] is then a [[derivative]].

$$
\dv{F_X(x)}{x} = f_X(x)
$$
