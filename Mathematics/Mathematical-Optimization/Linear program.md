A **linear program** is a program of the form

\begin{align}
&\min_{\vec{x}} \vec{c}^\top \vec{x} \\\\\ &\text{ s.t. } A\vec{x} \preccurlyeq \vec{b}
\end{align}

Standard Form
: \begin{align}
&\min_{\vec{x}} \vec{c}^\top \vec{x} \\\\\ &\text{ s.t. } A\vec{x} = \vec{b}, \vec{x} \preccurlyeq 0
\end{align}
