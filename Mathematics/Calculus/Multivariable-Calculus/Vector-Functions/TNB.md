# Introduction

## Definitions
\begin{equation}
\vec{T}(t) = \hat{\dot{r}}
\end{equation}

\begin{equation}
\vec{N}(t) = \hat{\dot{T}}
\end{equation}

\begin{equation}
\vec{B}(t) = \vec{T}(t)\times\vec{N}(t)
\end{equation}

### Application to Motion

\begin{equation}
\vec{a} = v' \vec{T} + \kappa v^2 \vec{N}
\end{equation}
