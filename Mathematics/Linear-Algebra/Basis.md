A **basis** $\mathcal{B}$ for a [[vector space]] is a minimal set of elements $\left\{ \vec{b}\_i\\}_{i \in I}$ needed to express the entirety of the space (using linear combinations). Every basis will have the same number of elements $n$, which defines the dimension of the vector space.

\begin{equation}
\dim V \triangleq | \mathcal{B} |
\end{equation}


In other words, any $\vec{v} \in V$ can be written as

\begin{equation}
\vec{v} = \sum_{i=1}^{\dim V} a_i \vec{b}_i
\end{equation}
