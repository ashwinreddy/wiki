**RISC-V** is a simplified [[instruction set architecture]] for computers.

RISC-V has 32 integer registers.

# Instructions

|Type|Op|
|---|-|
|R|Register-register|
|I|Short immediate data and load|
|S|Fetch store|
|B|Conditional jump|
|U|Long immediate|
|J|Unconditional jump|
