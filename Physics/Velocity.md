**Velocity** is the vector describing the rate at which a system moves relative to a reference frame.

In one-dimension

\begin{equation}
v(t) = \frac{\mathrm{d}x}{\mathrm{d}t} \equiv \dot{x}
\end{equation}

so that a vector velocity is

\begin{equation}
\vec{v}(t) = \dot{x}\mathbf{\hat{i}}+\dot{y}\mathbf{\hat{j}}+\dot{z}\mathbf{\hat{k}}
\end{equation}
