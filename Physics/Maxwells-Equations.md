**Maxwell's Equations** are a set of partial differential equations which describe how the electric and magnetic fields interact with each other.

# Formulations

## Standard Vector Calculus Form

|Number|Formula|
|------|:-------:|
|1|$\vec\nabla\cdot\vec{E} = \frac{\rho}{\epsilon_0}$|
|2|$\vec\nabla\times\vec{E} = -\frac{\partial B}{\partial t}$|
|3|$\vec\nabla\cdot\vec{B}=0$|
|4|$\vec\nabla\times\vec{B} = \mu_0 \vec{\jmath} + \mu_0\epsilon_0\frac{\partial \vec E}{\partial t}$|

## Single Equation

$$
\Box^2 A^\nu = -\mu_0 J^\nu
$$

where $A$ is the [Four-vector potential](./Four-Vector#examples_4-vector-potential)
